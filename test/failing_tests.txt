# Yosys upstream tests use `module ref (...)` — `ref` is a reserved
# keyword in IEEE 1800-2017 (function-arg direction).  Surelog
# correctly rejects it; Yosys's verilog frontend doesn't tokenize
# `ref` as a keyword so it accepts the file.  Covered locally by
# test/load_and_derive/ and test/resolve_types/ with `ref` renamed
# to `ref_mod`.
svinterfaces/load_and_derive.sv
svinterfaces/resolve_types.sv
