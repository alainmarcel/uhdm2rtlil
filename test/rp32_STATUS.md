# rp32 per-module test status

Imported from jeras/rp32 (see test/rp32/IMPORT_README.md).  Per-module tests are
generated by `test/import_design.py` (reusable for other imported designs):

    python3 test/import_design.py --design test/rp32 --prefix rp32 --mode uhdm-only \
      --skip-glob '*/fpga/*' --skip-glob '*/soc/*' \
      --skip-glob '*__xilinx_xpm*' --skip-glob '*__gowin*'

Each `rp32_<module>/project.f` lists the topologically-ordered package deps + the
module (relative paths into the shared `../rp32/`).  The Yosys *verilog* frontend
cannot parse the SV packages, so the tests are `# mode: uhdm-only`; verification
is the Verilator co-sim (UHDM netlist vs RTL), which works after the packed-struct
compat fixes in test/rp32/.
